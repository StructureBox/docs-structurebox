リップ形鋼(LC)の断面性能を取得する

POST https://api.structurebox.tech/v1/post_section/lc


このエンドポイントは、指定された リップ形鋼(LC) の寸法に基づいて、断面性能を取得します。

概要
  • このエンドポイントは、指定された リップ形鋼(LC) の寸法に応じて、断面性能を計算し、その結果を返します。
  • ボディパラメータ: H, B, C, t の寸法を送信します。
  • 結果として、断面積、断面二次モーメント、断面係数などが返されます。

ボディパラメータ​

パラメータ型説明制限
Hfloatリップ形鋼の高さ（mm）必須、数値のみ
Bfloatフランジ幅（mm）必須、数値のみ
Cfloatリップ幅（mm）必須、数値のみ
tfloat厚さ（mm）必須、数値のみ

入力例​

{
  "H": 100,
  "B": 50,
  "C": 5,
  "t": 7.5
}
注意

各パラメータはリップ溝形鋼のJIS登録寸法を指定する必要があります。入力ミスや無効な値が含まれている場合、400エラー（Bad Request）が返されます。

レスポンス形式​

成功した場合、以下の形式のレスポンスが返されます。

フィールド型説明
形状stringリップ形鋼の寸法情報
Afloat断面積（cm²）
Ixfloat断面二次モーメント（X軸方向, cm⁴）
Iyfloat断面二次モーメント（Y軸方向, cm⁴）
Zxfloat断面係数（X軸方向, cm³）
Zyfloat断面係数（Y軸方向, cm³）

iyfloat断面二次半径（Y軸方向, cm）
