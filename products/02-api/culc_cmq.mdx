# CMQを計算する 

<Highlight color="#1877F2">POST</Highlight> `https://api.structurebox.tech/v1/cmq/{type_model}`

<br />

**このエンドポイントは、与えられた `type_model` に基づいて、異なる計算モデルを適用し、せん断力、曲げモーメントなどのCMQを計算します。**


:::tip[概要]

- このエンドポイントは、指定された `type_model` に応じて異なる荷重項を計算し、その結果を返します。
- **パスパラメータ**: `type_model` は計算モデルの種類を指定します。
- **ボディパラメータ**: 各モデルに応じた入力データを送信します。

:::

### パスパラメータ

| パラメータ   | 型    | 説明                                   |
| ----------- | ----- | -------------------------------------- |
| `type_model` | `str` | 使用する計算モデルの種類。例：`type_point`, `type_zone` など |

---

### `type_point` モデル

- **説明**: 任意の位置に作用する集中荷重を計算します。
- **入力パラメータ**:
  - `al` (m): 部材長
  - `p` (kN): 荷重の大きさ（集中荷重）
  - `a` (m): 荷重が作用する位置

```json
{
  "al": 6.0,
  "p": 10.0,
  "a": 3.0
}
```

---

### `type_zone` モデル

- **説明**: 任意の区間に作用する等分布荷重を計算します。荷重が部材の左端から右端に均等または不均等に分布している場合に使用します。
- **入力パラメータ**:
  - `al` (m): 部材長
  - `a` (m): 荷重開始位置
  - `b` (m): 荷重の作用長さ
  - `w` (kN/m): 等分布荷重の強度
  - `awl` (m): 荷重左端の高さ
  - `bwl` (m): 荷重右端の高さ

```json
{
  "al": 6.0,
  "a": 1.0,
  "b": 3.0,
  "w": 10.0,
  "awl": 1.5,
  "bwl": 0.0
}
```

---

### `type_rect_full` モデル

- **説明**: 部材全長にわたる等分布荷重を計算します。
- **入力パラメータ**:
  - `al` (m): 部材長
  - `w` (kN/m): 等分布荷重の強度
  - `wl` (m): 負担幅

```json
{
  "al": 6.0,
  "w": 10.0,
  "wl": 1.5
}
```

---

```json
  "wl": 1.5

```

---

### `type_tri_right_full` モデル

- **説明**: 部材全長にわたって右端が高い三角形の等分布荷重を計算します。
- **入力パラメータ**:
  - `al` (m): 部材長
  - `w` (kN/m): 等分布荷重の強度

```json

  "wl": 1.5
}
```

## レスポンス形式

すべての `type_model` に共通のレスポンス形式です。

| フィールド  | 型       | 説明                         |
| --------- | -------- | ---------------------------- |
| `Ci`      | `float`  | 部材の左端せん断力（kN）     |
| `Cj`      | `float`  | 部材の右端せん断力（kN）     |
| `M0`      | `float`  | 部材の中央における曲げモーメント（kN・m） |
| `Qi`      | `float`  | 左端におけるせん断力（kN）   |
| `Qj`      | `float`  | 右端におけるせん断力（kN）   |

## エラーレスポンス

| ステータスコード | 説明                                                                                   |
| ---------------- | -------------------------------------------------------------------------------------- |
| `400 Bad Request` | 無効な `type_model` または、入力データのバリデーションエラーが発生した場合に返されます。 |
