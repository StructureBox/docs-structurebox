# リップ形鋼(LC)の断面性能を取得する

<Highlight color="#1877F2">POST</Highlight> `https://api.structurebox.tech/v1/post_section/lc`

<br />

**このエンドポイントは、指定された リップ形鋼(LC) の寸法に基づいて、断面性能を取得します。**

:::tip[概要]

- このエンドポイントは、指定された リップ形鋼(LC) の寸法に応じて、断面性能を計算し、その結果を返します。
- **ボディパラメータ**: `H`, `B`, `C`, `t` の寸法を送信します。
- 結果として、断面積、断面二次モーメント、断面係数などが返されます。

:::

## ボディパラメータ

| パラメータ | 型       | 説明                     | 制限                         |
| ---------- | -------- | ------------------------ | ---------------------------- |
| `H`        | `float`  | リップ形鋼の高さ（mm）     | 必須、数値のみ               |
| `B`        | `float`  | フランジ幅（mm）          | 必須、数値のみ               |
| `C`        | `float`  | リップ幅（mm）            | 必須、数値のみ               |
| `t`        | `float`  | 厚さ（mm）                | 必須、数値のみ               |

### 入力例

```json
{
  "H": 100,
  "B": 50,
  "C": 5,
  "t": 7.5
}
```

:::warning[注意]

各パラメータはリップ溝形鋼のJIS登録寸法を指定する必要があります。入力ミスや無効な値が含まれている場合、400エラー（Bad Request）が返されます。<br />

:::

## レスポンス形式

成功した場合、以下の形式のレスポンスが返されます。

| フィールド | 型       | 説明                                 |
| ---------- | -------- | ------------------------------------ |
| `形状`     | `string` | リップ形鋼の寸法情報                 |
| `A`        | `float`  | 断面積（cm²）                       |
| `Ix`       | `float`  | 断面二次モーメント（X軸方向, cm⁴）  |
| `Iy`       | `float`  | 断面二次モーメント（Y軸方向, cm⁴）  |
| `Zx`       | `float`  | 断面係数（X軸方向, cm³）            |
| `Zy`       | `float`  | 断面係数（Y軸方向, cm³）            |
| `ix`       | `float`  | 断面二次半径（X軸方向, cm）         |
| `iy`       | `float`  | 断面二次半径（Y軸方向, cm）         |
| `Cy`       | `float`  | Cy（Y軸方向のせん断中心, cm）       |

## エラーレスポンス

| ステータスコード | 説明                                                                                   |
| ---------------- | -------------------------------------------------------------------------------------- |
| `404 Not Found`   | 指定された寸法に一致するリップ形鋼が見つからない場合に返されます。                       |
| `400 Bad Request` | 入力データが無効な場合、またはバリデーションエラーが発生した場合に返されます。           |
