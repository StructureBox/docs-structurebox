# 溝形鋼(WC)の断面性能を取得する

<Highlight color="#1877F2">POST</Highlight> `https://api.structurebox.tech/v1/steel/post_section/wc`

<br />

**このエンドポイントは、指定された 溝形鋼(WC) の寸法に基づいて、断面性能を取得します。**

:::tip[概要]

- このエンドポイントは、指定された 溝形鋼(WC) の寸法に応じて、断面性能を計算し、その結果を返します。
- **ボディパラメータ**: `H`, `B`, `tw`, `tf` の寸法を送信します。
- 結果として、断面積、断面二次モーメント、断面係数などが返されます。

:::

## ボディパラメータ

| パラメータ | 型       | 説明                     |
| ---------- | -------- | ------------------------ |
| `H`        | `float`  | 溝形鋼の高さ（mm）       |
| `B`        | `float`  | フランジ幅（mm）         |
| `tw`       | `float`  | ウェブ厚さ（mm）         |
| `tf`       | `float`  | フランジ厚さ（mm）       |

## 入力例

```json
{
  "H": 300,
  "B": 90,
  "tw": 9,
  "tf": 13
}
```

:::warning[注意]

各パラメータは溝形鋼のJIS登録寸法を指定する必要があります。入力ミスや無効な値が含まれている場合、400エラー（Bad Request）が返されます。<br />

:::

## レスポンス形式

成功した場合、以下の形式のレスポンスが返されます。

| フィールド | 型       | 説明                                |
| ---------- | -------- | ----------------------------------- |
| `形状`     | `string` | 溝形鋼の寸法情報                    |
| `A`        | `float`  | 断面積（cm²）                       |
| `Ix`       | `float`  | 断面二次モーメント（X軸方向, cm⁴）  |
| `Iy`       | `float`  | 断面二次モーメント（Y軸方向, cm⁴）  |
| `Zx`       | `float`  | 断面係数（X軸方向, cm³）            |
| `Zy`       | `float`  | 断面係数（Y軸方向, cm³）            |
| `ix`       | `float`  | 断面二次半径（X軸方向, cm）         |
| `iy`       | `float`  | 断面二次半径（Y軸方向, cm）         |
| `Cy`       | `float`  | Cy（Y軸方向のせん断中心, cm）       |

## エラーレスポンス

| ステータスコード | 説明                                                                                   |
| ---------------- | -------------------------------------------------------------------------------------- |
| `404 Not Found`   | 指定された寸法に一致する溝形鋼が見つからない場合に返されます。                           |
| `400 Bad Request` | 入力データが無効な場合、またはバリデーションエラーが発生した場合に返されます。           |
