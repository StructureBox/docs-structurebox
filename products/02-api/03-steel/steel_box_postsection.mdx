# 正方形鋼管(BOX)の断面性能を取得する

<Highlight color="#1877F2">POST</Highlight> `https://api.structurebox.tech/v1/steel/post_section/box`

<br />

**このエンドポイントは、指定された 正方形鋼管(BOX) の寸法に基づいて、断面性能を取得します。**

:::tip[概要]

- このエンドポイントは、指定された 正方形鋼管(BOX) の寸法に応じて、断面性能を計算し、その結果を返します。
- **ボディパラメータ**: `B`, `t` の寸法を送信します。
- 結果として、断面積、断面二次モーメント、断面係数などが返されます。

:::

## ボディパラメータ

| パラメータ | 型       | 説明                     |
| ---------- | -------- | ------------------------ |
| `B`        | `float`  | 正方形鋼管の外径（mm）     |
| `t`        | `float`  | 厚さ（mm）                |

#### 入力例

```json
{
  "B": 150,
  "t": 6
}
```

:::warning[注意]

正方形断面のみサポートされています。長方形のBOX断面は対応しておりませんのでご注意ください。

:::

---

## レスポンス形式

成功した場合、以下の形式のレスポンスが返されます。

| フィールド | 型       | 説明                                 |
| ---------- | -------- | ------------------------------------ |
| `形状`     | `string` | 正方形鋼管の寸法情報                 |
| `A`        | `float`  | 断面積（cm²）                       |
| `Ix`       | `float`  | 断面二次モーメント（X軸方向, cm⁴）  |
| `Iy`       | `float`  | 断面二次モーメント（Y軸方向, cm⁴）  |
| `Zx`       | `float`  | 断面係数（X軸方向, cm³）            |
| `Zy`       | `float`  | 断面係数（Y軸方向, cm³）            |
| `ix`       | `float`  | 断面二次半径（X軸方向, cm）         |
| `iy`       | `float`  | 断面二次半径（Y軸方向, cm）         |

---

## エラーレスポンス

| ステータスコード | 説明                                                                                   |
| ---------------- | -------------------------------------------------------------------------------------- |
| `404 Not Found`   | 指定された寸法に一致する正方形鋼管が見つからない場合に返されます。                       |
| `400 Bad Request` | 入力データが無効な場合、またはバリデーションエラーが発生した場合に返されます。           |
